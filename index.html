Rollercoaster Modeler

Controls:

  • Mouse Drag: Rotate view
  • Mouse Wheel: Zoom
  • 1-4: Snap to viewpoint (Front, Top, Side, Isometric)
  • A: Add vertex at random position
  • D: Delete selected vertex
  • C: Cycle curve types
  • Arrows: Move selected vertex (Ctrl/Cmd for Z-axis)
  • Shift+Arrows: Move faster
  • R: Reset scene

Click vertices to select them. Edit coordinates in the HUD panel on the left.
